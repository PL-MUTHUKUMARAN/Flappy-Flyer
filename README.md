# 🐦 Flappy Flyer

A fun SVG-based version of the classic Flappy Flyer game, built using **HTML**, **CSS**, and **Vanilla JavaScript**. The game features smooth animations, sound effects, scoring, collision detection, and mobile/keyboard controls — all rendered inside an SVG canvas.

## 🎮 Features

- ✅ Simple and clean SVG-based game design  
- ✅ Responsive for desktop and mobile  
- ✅ Collision detection with pipes and ground  
- ✅ Smooth wing animation and gravity physics  
- ✅ Score tracking and best score saved using `localStorage`  
- ✅ Restart with countdown timer and restart button  
- ✅ Sound effects: jump, death, countdown  
- ✅ Works with both **keyboard** and **touch** controls  

---

## 🚀 How to Play

1. Clone the repo or [Download ZIP](https://github.com/PL-MUTHUKUMARAN/Flappy-Flyer/archive/refs/heads/main.zip)  
2. Open `index.html` in any modern web browser  
3. Tap the screen or press any key to flap  
4. Avoid pipes and the ground  
5. Try to beat your best score!
